== Description ==

Vina Camera Slideshow Widget presents excellent interface that hooks the viewer and shoots up the popularity of your website. It is an ultimate and light weight Slider module. This module is cross browser compatible and works on all the modern browsers. The module is built using jQuery, it’s very easy to install and configure with lot many features.

Features:

- Support multi Wordpress Version.
- Zoom in and Zoom out images.
- Load image from Featured Image.
- Get title, image, description from post item.
- Hardware accelerated: using CSS3 for supported iOS, Android and WebKit browsers.
- FF/Safari/Chrome/IE7+: Full modern browser support for desktop.
- Customizable speed, delay, size etc.
- Languages included: English
- Allow loading wordpress widget inside post, page.
- To manipulate almost everything !
- Totally customizable and skinnable using HTML and CSS
- Lightweight, modern and fast-loading design.
- W3C XHTML 1.0 Transitional. W3C CSS Valid.
- Tableless design and 100% CSS-based
- Fully compatible IE7+, Firefox 2+, Flock 0.7+, Netscape, Safari, Opera 9.5
- Easy installation

== Installation ==

This section describes how to install the plugin and get it working.

e.g.

1. Upload the entire 'vina-camera-widget' folder to the '/wp-content/plugins/' directory
2. Activate the plugin through the 'Plugins' menu in WordPress
3. Add the 'Vina Camera Slideshow Widget' to your sidebar
